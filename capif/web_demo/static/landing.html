API Lifetime Flow

1

Publish API

The Operator registers via mTLS (APF cert) and publishes the SIM Swap API to the CAPIF catalog.

POST /registrationsPOST /service-apis
2

Discover & Authorize

The Bank registers, discovers the cataloged API, and requests an OAuth2 access token from CAPIF.

POST /onboardedInvokersGET /allServiceAPIsPOST /token
3

Consume & Protect

The Bank calls the SIM Swap API; the AEF verifies the token signature and logs the invocation.

POST /sim-swap/checkPOST /api-invocation-logs
4

Audit

The Operator (AMF cert) audits CAPIF to see who invoked the API and when.

GET /logs/apiInvocationLogs
